## 4개 코스 — 우선순위와 시간 추정

| 순서 | 코스 | 분량 | 시간 추정 | 난이도 | 비고 |
|---|---|---|---|---|---|
| 1 | Introduction to Agent Skills | 6 챕터 | 1~1.5h | ★ | 워밍업, 가장 짧음 |
| 2 | Claude Code in Action | 4 섹션 | 1.5~2h | ★★ | Claude Code 사용 경험 있으면 빠름 |
| 3 | Introduction to MCP | 4 섹션 | 2~3h | ★★★ | Python+JSON 기초 필요. 평가 있음 |
| 4 | Building with Claude API | 9 모듈 | 4~6h | ★★★★ | 보스. 마지막 집중시간 확보 |

**총 9~13시간** — 하루 풀로 빼서 끝내거나, 2일 분산 권장.

## 코스별 학습 목표 (요약)

### 1. Intro to Agent Skills (6 챕터)
1. What are skills?
2. Creating your first skill
3. Configuration and multi-file skills
4. Skills vs. other Claude Code features
5. Sharing skills
6. Troubleshooting skills

### 2. Claude Code in Action (4 섹션)
- Claude Code란? / 실습 시작 / 고급 기능(MCP, GitHub 통합) / 훅·SDK + 퀴즈

### 3. Intro to MCP (4 섹션)
- Introduction
- Hands-on with MCP servers (도구 정의, 서버 검사기)
- Connecting with MCP clients (리소스, 프롬프트)
- Assessment & wrap up
- 핵심 3요소: Tools / Resources / Prompts

### 4. Building with Claude API (9 모듈)
1. Introduction
2. Accessing Claude with the API (인증, 멀티턴)
3. Prompt Evaluation (테스트셋, 채점)
4. Prompt Engineering Techniques
5. Tool Use with Claude
6. RAG and Agentic Search (청킹, 임베딩, BM25)
7. Features of Claude (확장 사고, 이미지/PDF, 캐싱)
8. Model Context Protocol
9. Agents and Workflows (병렬, 체이닝, 라우팅)
